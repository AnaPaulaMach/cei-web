# Guia visual CEI OS

## Concepto

CEI OS es la identidad visual del sitio: un mini sistema operativo de la Comision de Estudiantes de Informatica. Debe sentirse como un escritorio retro universitario, con ventanas, barra de tareas, accesos directos, paneles de sistema, terminales y detalles pixelados.

La referencia es Windows XP / Minecraft / informatica retro, pero adaptada a CEI. No se copian marcas, logos ni interfaces exactas.

## Paleta inicial

- Azul CEI: `#1d4ed8`
- Azul cielo: `#38bdf8`
- Verde terminal: `#22c55e`
- Gris ventana: `#e9f2ff`
- Azul sombra: `#09245f`
- Amarillo alerta: `#facc15`
- Blanco papel: `#ffffff`
- Negro terminal: `#07111f`

La paleta debe sentirse tecnologica y clara, no solo azul. Usar verde terminal y amarillo alerta para estados, llamados o badges.

## Tipografias

- Texto largo: sans legible del sistema.
- Titulos, badges y pequenos labels: monospace o pixel-like.
- Evitar usar fuente pixel para parrafos largos o formularios.

## Componentes

- Ventanas: bordes marcados, barra superior azul, botones pequenos tipo minimizar/maximizar/cerrar.
- Botones: relieve clasico, foco visible, estados hover claros.
- Tarjetas: pueden parecer archivos, accesos directos o paneles de sistema.
- Iconos: usar estilo pixel/desktop, simples y reconocibles.
- Formularios: priorizar legibilidad, labels claros y buen espaciado.
- Barra de tareas: puede aparecer como elemento persistente o decorativo, sin tapar contenido.

## Tono visual

Joven, informatico, cercano y universitario. La estetica puede tener humor visual, pero la informacion institucional debe seguir siendo seria.

## Evitar

- Que la estetica retro vuelva dificil leer o completar formularios.
- Usar demasiadas animaciones al mismo tiempo.
- Copiar logos, textos o interfaces exactas de marcas registradas.
- Depender solo de color para transmitir estados.
- Ocultar links importantes solo en el footer.
